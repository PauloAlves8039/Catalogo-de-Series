<h1 align="center">:tv: Catalogo de Series</h1>

## :computer: Projeto
Repositório de uma aplicação web para catalogar uma lista de series favoritas usando MEVN Stack.

- Bootstrap-vue v2.12.0
- MongoDb v3.4.10
- Express v4.17.1
- Vue.Js v2.9.11
- Node.Js v12.16.2
- Vue Router v3.1.6
- Vuex v3.1.3
- Vue CLI v4.2.3
- Axios v0.19.2
- Eslint v6.7.2
