<h1 align="center">:tv: Catalogo de Series</h1>

## :computer: Projeto
Repositório de uma aplicação web para catalogar uma lista de series favoritas usando MEVN Stack.

## :wrench: Recursos Utilizados
- MongoDB
- Express
- Vue.Js
- Node.Js
